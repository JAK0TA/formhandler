
.. _valuemax:

========
ValueMax
========

Checks if the value of a field is less or equal than the configured value.

..  code-block:: typoscript

    Example Code:

    validators {
      DefaultValidator {
        model = DefaultValidatorModel
        config {
          fields {
            age.errorChecks {
              valueMax {
                model = ValueMaxModel
                valueMax = 100
              }
            }
            mass.errorChecks {
              valueMax {
                model = ValueMaxModel
                valueMax = 10.99
              }
            }
          }
        }
      }
    }
