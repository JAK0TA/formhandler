
.. _float:

=====
Float
=====

Checks if a field contains a valid float value.

..  code-block:: typoscript

    Example Code:

    validators {
      DefaultValidator {
        model = DefaultValidatorModel
        config {
          fields {
            mass.errorChecks {
              float {
                model = FloatModel
              }
            }
          }
        }
      }
    }
