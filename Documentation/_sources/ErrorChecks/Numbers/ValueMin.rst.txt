
.. _valuemin:

=========
ValueMin
=========

Checks if the value of a field is at least the configured value.

..  code-block:: typoscript

    Example Code:

    validators {
      DefaultValidator {
        model = DefaultValidatorModel
        config {
          fields {
            age.errorChecks {
              valueMin {
                model = ValueMinModel
                valueMin = 18
              }
            }
            mass.errorChecks {
              valueMin {
                model = ValueMinModel
                valueMin = 1.5
              }
            }
          }
        }
      }
    }
