
.. _integer:

=======
Integer
=======

Checks if a field contains a valid integer value.

..  code-block:: typoscript

    Example Code:

    validators {
      DefaultValidator {
        model = DefaultValidatorModel
        config {
          fields {
            age.errorChecks {
              integer {
                model = IntegerModel
              }
            }
          }
        }
      }
    }
