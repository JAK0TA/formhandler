
.. _tablelookup:

===========
TableLookup
===========

Checks if the value of a field is or is not in a configured field in a configured table.

..  code-block:: typoscript

    Example Code:

    validators {
      DefaultValidator {
        model = DefaultValidatorModel
        config {
          fields {
            email.errorChecks {
              tableLookup {
                model = TableLookupModel
                table = fe_users
                field = email
                exists = False
                excludeHidden = True
                additionalWhere = FIND_IN_SET(1, usergroup)
              }
            }
          }
        }
      }
    }
